# SmartSpend - AI Powered Expense Tracker

## Tech Stack
- Frontend: React (Vite), TailwindCSS, Axios, React Router, Recharts
- Backend: Node.js, Express, MongoDB (Mongoose), JWT
- AI Insights: Rule-based intelligent suggestions from spending behavior

## Project Structure
- client/
- server/

## Setup Instructions

### 1) Backend
1. Go to `server`
2. Install dependencies: `npm install`
3. Create env from `env.example` and configure values
4. Run: `npm run dev`

### 2) Frontend
1. Go to `client`
2. Install dependencies: `npm install`
3. Run: `npm run dev`

Frontend: `http://localhost:5173`
Backend: `http://localhost:5000`

## Core Features Implemented
- JWT authentication with protected routes
- Onboarding flow storing financial profile
- Expense CRUD with filters
- Smart dashboard (cards + pie chart + line chart)
- Personalized AI insights and mini monthly calculator
