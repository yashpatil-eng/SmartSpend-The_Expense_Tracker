# SmartSpend – Intelligent Expense Tracking & Analysis System

## Project Description
SmartSpend is a web-based application to track, categorize, and analyze daily expenses.
It provides monthly summaries, category-wise insights, and budget alerts.

## Features
- Add daily expenses
- Expense categorization
- Monthly & yearly reports
- Budget alerts
- Interactive charts

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask / Node.js / Java
- Database: MySQL / MongoDB
- Version Control: Git & GitHub
